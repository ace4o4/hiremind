1. Vision & Mission

Vision: Every job seeker (especially fresh graduates and career switchers in India and globally) should walk into any interview feeling 100% prepared, confident, and zero anxiety — because they have practiced with a personal AI Guru that knows them better than any human coach.

Mission: Build the world’s most advanced agentic, long-memory, emotionally intelligent mock interview platform that simulates real-world interviews so accurately that users report “it felt more real than my actual interview.”

Tagline:
“Not just mock. Your forever Interview Guru who remembers everything and levels you up until you land the job.”

2. Executive Summary

InterviewForge is a premium agentic AI platform that delivers end-to-end, hyper-personalized, multi-modal mock interviews using 6 game-changing differentiators no competitor offers together in 2026:

Long-term Memory Mentor – Remembers every past session forever.

Live Company Intelligence – Pulls real-time company news and asks killer questions.

Panel Attack Mode – 3 distinct AI personalities interview you simultaneously and interact with each other.

Real-time Stress & EQ Coach – Detects anxiety live and gives subtle calming cues.

Replay & Level Up – Re-record any answer → instant side-by-side video comparison + score improvement.

Voice Cloning Feedback – Hears your perfect answer spoken in your own cloned voice.

Target launch: 10–12 weeks MVP.
Expected outcome: 15k+ active users in 6 months, 45%+ Pro conversion, NPS ≥ 80.

5. Full Agentic Interview Lifecycle (Detailed)

5.1 Pre-Interview (Max 6 minutes)

Inputs:

Resume (PDF/DOCX/Text)

JD (paste or LinkedIn URL)

Agents Activated:

Resume Parser Agent

JD Analyzer Agent

Gap Detector Agent

Company Intelligence Agent (Tavily/SerpAPI tool call → latest news, funding, layoffs, products, Glassdoor sentiment)

Outputs:

Visual Interview Blueprint (radar chart + question flow)

3-5 hyper-relevant company-specific questions auto-generated

Acceptance Criteria:

Supports 10+ languages for resume/JD

Accuracy > 92% on skill extraction (tested on 500 real resumes)

5.2 During Interview (Core Magic)

Modes Available:

Voice-only (default)

Video + Avatar (calm professional look, role-specific tone)

Panel Attack Mode (toggle): Strict HR + Tough Tech Lead + Friendly Manager on one screen. They can interrupt and talk to each other.

Multi-Agent System (LangGraph stateful graph):

Interviewer Agent (main speaker)

Evaluator Agent (scores in background)

Adaptor Agent (decides next question)

EQ Coach Agent (runs every 8-12 seconds: detects fillers, pitch rise, eye contact drop via MediaPipe + Hume-like model → subtle on-screen tip like “Breathe… you’re doing great”)

Memory Agent (pulls relevant past sessions in <800ms)

Technical Features:

Shared interactive coding canvas with real-time execution sandbox (Python/JS/Java supported)

Pause / Hint / Re-answer anytime

Auto-save every 30 seconds

Performance Requirements:

STT latency < 1.0s (Whisper-large-v3 or Groq)

Agent reasoning < 1.8s

Video sync < 300ms (LiveKit)

5.3 Post-Interview (Instant <15s)

Report Components (Beautiful, Exportable PDF + Web Dashboard):

Hero Overall Score (0-100) + Confidence Trend

Radar Chart (Content, Structure, Delivery, Non-verbal, Relevance)

Timestamped Highlight Reel (best 3 + worst 3 clips – clickable)

Anxiety Map – timeline heatmap of stress moments + breathing exercise links

Replay & Level Up – For every weak segment: “Re-do” button → record new → side-by-side video + score delta

Voice Cloning Feedback – AI writes best answer → ElevenLabs clones user’s voice → plays perfect version

Personalized 30-day Roadmap (5 specific exercises + sample answers)

Benchmark vs role peers (anonymous aggregate data)

Progress Hub (Lifetime View):

Skill growth curves (STAR mastery 42% → 89%)

Streak counter + motivational badges

Weekly AI email summary

8. Technical Architecture (Build-Ready)

Frontend: Next.js 15 (App Router) + Tailwind + shadcn/ui + Framer Motion + Zustand + TanStack Query
Backend: Python FastAPI (agents) + Node.js (real-time)
Agent Orchestration: LangGraph (stateful, persistent memory)
Memory: Supabase PostgreSQL + pgvector (per-user vector DB)
LLMs: Claude 4 Sonnet (reasoning) + Groq Llama-3.3-70B (speed) + fine-tuned interviewer persona
STT: Whisper-large-v3 (or Deepgram)
TTS & Voice Cloning: ElevenLabs (instant cloning + emotional control)
Video: LiveKit WebRTC
Emotion Detection: MediaPipe + Hume AI API fallback
Company Research: Tavily API (tool calling)
Database: Supabase (auth, storage, vectors)
Deployment: Vercel (frontend) + Railway/Fly.io (agents) + Docker
Monitoring: Sentry + PostHog + Prometheus

Folder Structure & Starter Repo Guidelines provided in Appendix A (full spec).
